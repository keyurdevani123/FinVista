# FinVista

AI-powered personal finance and investment manager built with Next.js.

## Repository Layout

This repository is intentionally organized as:

```text
/
	README.md
	FinVista/
		(full Next.js project source)
```

## Quick Start

```bash
git clone https://github.com/keyurdevani123/FinVista.git
cd FinVista/FinVista
npm install
```

Create `FinVista/.env` and set your environment variables.

Run locally:

```bash
npm run dev
```

## Tech Stack

- Next.js (App Router)
- React + Tailwind CSS + shadcn/ui
- Prisma + PostgreSQL
- Clerk Authentication
- Inngest + Resend
- Arcjet
